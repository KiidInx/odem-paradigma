# ODEM — Organic Learning Machine

**A new paradigm for local, persistent, growing intelligence.**

ODEM is not a Large Language Model. It is an **OLM — Organic Learning Machine**. Knowledge does not live in billions of frozen weights trained once in a datacenter. Knowledge grows organically, experience by experience, in a living memory structure called the Helix.

This repository contains the public whitepaper establishing prior art for the OLM paradigm, the nanoagent collective architecture, the Mindhive communication system, and the swarm-embedding method.
